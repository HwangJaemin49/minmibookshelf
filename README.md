 # 📚 1886 서재 (1886 Bookshelf)                                                                           
                                                                                                           
프라이빗 독서 모임 멤버들을 위한 **온라인 공유 책장** 서비스입니다. 함께 읽은 책들을 시각적인 책장 형태로  
아카이빙하고 관리할 수 있습니다.                                                                           
                                                                                                           
## ✨ 주요 기능                                                                                            
                                                                                                           
- **보안 입장 (Passcode Gate):** 모임원들만 아는 암호(`1886`)를 입력해야만 서재에 입장할 수 있습니다.      
- **가상 책장 UI:** 실제 나무 선반에 책이 꽂혀 있는 듯한 감성적인 인터페이스를 제공합니다.                 
- **도서 검색 및 추가:** 책 제목을 입력하여 검색하고, 마음에 드는 책을 즉시 책장에 추가할 수 있습니다.     
- **로컬 저장소 유지:** 추가된 도서 목록은 브라우저의 `LocalStorage`에 저장되어 재방문 시에도 유지됩니다.  
- **반응형 웹:** 모바일과 PC 환경 모두에서 최적화된 화면을 보여줍니다.                                     
                                                                                                           
## 🛠 사용 기술                                                                                            
                                                                                                           
- **Language:** HTML5, CSS3, JavaScript (Vanilla JS)                                                       
- **Styling:** CSS Grid & Flexbox, Google Fonts (Noto Serif KR)                                            
- **Persistence:** LocalStorage API                                                                        
                                                                                                           
## 🚀 시작하기                                                                                             
                                                                                                           
1. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.                                              
2. 입장 암호 창에 `1886`을 입력합니다.                                                                     
3. 책장의 도서를 확인하거나, 우측 하단의 `+` 버튼을 눌러 새 책을 추가해 보세요.
